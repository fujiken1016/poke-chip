# ポケチップ 🃏

仲間内ポーカー用のチップ管理＋カード配布アプリ。ログイン不要、ルームコード（4文字）で各自のスマホから参加。

## 2つのモード

| モード | 使う場面 | アプリがやること |
|-------|---------|--------------|
| **チップモード** | トランプはあるがチップがない | ベット・ポット・サイドポット・ブラインド・ボタン管理。勝敗判定は人間、ディーラーが勝者にポットを渡す |
| **フルモード** | トランプもない | 上記＋手札を各スマホに配布、ボード表示、役の自動判定・ポット自動分配（チョップ/サイドポット対応） |

## ディーラー役（🎩）

ルーム作成者がディーラー。プレイング可・専任（離席）も可。権限：
ハンド開始 / （チップモード）ストリート進行・ポット授与 / **1手ずつ巻き戻し（最大20手・履歴付き）** / **任意プレイヤーのベット修正** / **±ポット修正** / **手番プレイヤーの代理アクション（フォールド・チェック・コール）** / **＋追加（リバイ/アドオン・収支に計上）** / ±修正（誤り訂正・収支に計上しない）/ ブラインド変更 / 次ハンドのボタン指定 / ハンド中止（ベット返却）/ 🎩交代 / 離席・退出処理

## バイイン・収支

- 参加時のバイイン（持ち点）は**必須入力**（下限はBB）。ルーム作成者は「初期スタック」が自分のバイイン
- 「💰 収支」パネル（初期展開・全員に公開）：バイイン累計 / 現在 / ±収支。各シートにも±収支をミニ表示。退出者の精算記録も残る
- 「退出」ボタンで自分の収支を確認したうえで精算退出。ディーラーが退出した場合は次のプレイヤーに🎩を自動引き継ぎ

## ベットUI・進行

- クイックボタン：1/3ポット / 1/2ポット / ポット / オールイン（コール後ポット基準、最小レイズ・スタックでクランプ）＋手入力
- **アクションクロック**（デフォルト20秒・**ルーム作成時に5〜300秒で設定可**、サーバー強制）：残り時間は手番シートと自分のバナーに**横棒メーター**で表示、残り5秒で赤点滅。時間切れはチェック可能ならチェック、ベットされていればフォールド（ログに「時間切れ」）
- 手番になると効果音＋バイブ（対応端末）＋「YOUR TURN」表示

## オールイン対決（フルモード）

- ベット続行不能なオールイン対決になると全員の手札を公開し、**ランイット回数（1〜3回）を各自が選択**。全員選択で開始、**少ない回数が採用**（時間切れは1回扱い）
- ランごとに残りのボードを配り直し、ポットをラン数で分割して精算（サイドポット対応、端数は1回目に寄せる）
- オールイン以降は**エクイティ（勝率%）を全員に表示**。ストリートが進むたびに更新され、リバーで100%/0%に確定
- ショーダウン結果は**役の詳細**（例: ツーペア（A・K）、フラッシュ（Qハイ））付き。参加者全員の役を明記（マックした人は表示しない）

## BOT（コンピュータープレイヤー）

- ディーラーが「🤖 BOT追加」で好きな人数（席が空いている限り）追加できる（フルモード専用）
- BOTは自分の手札とボードだけから勝率をモンテカルロ推定して行動（他人の手札は見ない）。1〜2秒考えてからチェック/コール/レイズ/フォールド、たまにブラフ。ランイット選択も自動
- 退出させたい時はディーラーパネルから「退出」

## 通信

- リアルタイム同期はSSE。**SSEが届かない環境（Cloudflareトンネル等）では自動で1.2秒ポーリングに切替**（無操作で全機能動作）

## 観戦者モード

- 参加画面の「👁 観戦者として参加」でバイイン不要で入室。プレイはできない
- 観戦者には**全員の手札が常時公開**され、**各プレイヤーのエクイティがリアルタイム表示**（アクション・ストリートごとに変化）
- プレイヤー側には観戦者の存在数のみ表示（手札は漏れない）

## テーブルUI

- 楕円ポーカーテーブル表示（自分が下中央）。各シートに：カード裏面（ハンド参加中の表現）／ネームプレート（スタック＋±収支）／直近アクションのタグ（チェック・コール・レイズ・フォールド・ALL IN）／SB・BBバッジ／手番クロック。卓上にベット額チップ・Dボタン・ポット・ボード
- プレイヤーの公開ID（pub）と認証用秘密ID（pid）を分離。他人の秘密IDは配信されないため、なりすまし操作は不可

## 演出・サウンド

- カットイン演出：FOLD/CHECK/CALL/RAISE/ALL IN/WIN/フェーズ進行
- 効果音（WebAudio合成・素材ファイル不要）：チップ音・チェックのノック音・フォールド・ALL IN・WIN・手番通知。ヘッダー🔊でミュート切替（端末ごとに記憶）
- ルーム名は作成時に自由入力（空欄なら「◯◯の卓」）。ヘッダーにルーム名＋参加コード表示、プレイヤー一覧にオンライン状態（🟢）と人数

## 起動

```sh
node server.js   # Node 18+（依存パッケージなし）。ポートは PORT 環境変数（デフォルト 3300）
```

- **同じ場所で遊ぶ**（メインの使い方）: Macで起動 → 同じWi-Fiのスマホから `http://<MacのローカルIP>:3300` を開く
- **旅先などMacがない場合**: Render / Fly.io / Railway 等の無料Nodeホスティングにこのフォルダを置くだけで動く（`server.js` は依存ゼロ・単一プロセス。状態はメモリ保持なのでスリープしない有料枠かプレイ中だけ起きてればOK）

## 実装メモ

- `server.js`: 静的配信＋JSON API＋SSE（リアルタイム同期）。テキサスホールデムのベッティングエンジン（ミニマムレイズ、ショートオールインの再レイズ制限、コールされなかったベットの返却、サイドポット計算、端数チップはボタン左隣から分配）
- `public/`: バニラJS・スマホ最適化。手札は自分にだけ配信（サーバー側でフィルタ）。再接続はタブ単位（sessionStorage）
- ルーム状態はメモリのみ（サーバー再起動で消える）
